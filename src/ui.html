How it works

The Color Palette Generator Plugin allows users to create color palettes based on a base color in Figma. By simply entering the hex value and name of a color, the plugin generates a complete set of color primitives ranging from 50 (lightest) to 950 (darkest), using the entered color as the "500" base.

Steps

01

Enter Your Color

Enter a color’s hex value (e.g., #FF5733) and provide a name for the color.

02

Palette Creation

The plugin calculates and generates a color palette with shades and tints from 50 to 950, based on the base color you input.

03

Instant Variables

The plugin directly creates Figma variables for each of the generated colors within Figma's variable collection, ready for use in your designs.
